# projeto-algoritmos
Repositorio feito com o intuito de gerenciar o trabalho da prova de Algoritmos da UNISAGRADO em grupo


Para rodar o projeto:

Certifique-se de ter as seguintes bibliotecas instaladas com o python:

- peewee
- cryptography
- tkinter
- pillow

Após todas as instalações necessárias, abra o terminal do visual studio code com a pasta do projeto selecionada e execute o seguinte comando:

python main.py

Se tudo ocorrer bem, você já estará executando o projeto perfeitamente!

Na sua primeira sessão, um cadastro deverá ser feito para que os arquivos necessários sejam criados.
