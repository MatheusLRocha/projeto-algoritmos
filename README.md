# projeto-algoritmos
Repositorio feito com o intuito de gerenciar o trabalho da prova de Algoritmos da UNISAGRADO em grupo


Para rodar o projeto:

Crie uma pasta venv no projeto para evitar conflitos globais nas bibliotecas da sua máquina, de preferência na versão 3.12 do python:

python -m venv .venv - para baixar na versão atual que se tem na máquina

ou

py -3.12 -m venv .venv - caso tenha o python 3.12 baixado na máquina

ative o ambiente virtual de acordo da forma que sua máquina pede

Certifique-se de ter as seguintes bibliotecas instaladas dentro da .venv com o python:

- peewee: pip install peewee
- cryptography: pip install cryptography
- tkinter: pip install tk


Após todas as instalações necessárias, no terminal do visual studio code com a pasta do projeto selecionada e a venv ativada, execute o seguinte comando:

python main.py

Se tudo ocorrer bem, você já estará executando o projeto perfeitamente!

A senha para acesso é admin.
